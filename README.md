# ktcpvs-matrix 简介

Stands for Kernel TCP Virtual Server, 在[KTCPVS][]基础上开发，可以在 hosts-apps两个维度上设置权值

读书的时候，导师的课题有需求，所有就有了这个软件，写于2008年，基于红帽子rhel4.0(linux-2.6.9)，整理硬盘翻出来的，也许有人会需要

## 简单介绍

  * [demo/eapac.exe][demo] 演示录像
  * script/*       脚本工具
  * userspace/     用户工具
